If your suggested plugin is a mod, we are not interested because mods are not compatible with multiplayer servers. For example, Mo' Creatures is a mod, not a plugin because it adds mobs, textures, and sounds to the game. World Edit, however, would be considered a plugin because it does not add any of these or change the original Minecraft code.

Examples of Mods:
Mo' Creatures Mod
Aether Mod
Herobrine Mod (Herobrine No Longer Exists in the Game)
Too Many Items
Humans+
Millenear
Recipe Book

Examples of Plugins:
World Edit
Essentials
World Guard
Factions
Mob Rider
Voxel
